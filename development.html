Development


Assignment Goals

  • Develop an interactive interface in React
  • Use interface components that have an internal data state
  • Specifically, design a list-based interface that allows users to select items out of a list
  • Aggregate selected items and sort/filter them

Interface Description

I created an example bakery shopping cart mockup! It shows different items you can purchase and allows for filtering/sorting of items, as well as an interactive shopping cart.

Interface Architecture

  • Filters to sort by (vegan/non-vegan, bready pastries/non-bready pastries)
  • Sort items by price
  • Interactive shopping cart that allows you to add and remove items
  • All combinations of filters/sort work together
  • Reset button to undo all filters/sorts
